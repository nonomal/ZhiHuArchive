上面这篇文章介绍了 Mathpix 的 API 获取方式，但是调用这个 API 还需要自己写代码，并不方便。

用 Quicker 动作来做这个 API 的前端可以省去大量操作。更何况，已经有人把这个动作做出了并公开了：

截图识别公式 - 已分享的动作 - Quicker

所以，简单梳理一下，具体的流程是：

  1. 访问 https://dashboard.mathpix.com 并注册账号
  2. 输入信用卡信息，不需要Visa，保证可用额度在 8 元以上，用于验证扣款（会打回来）
  3. 启动 Quicker，打开截图识别公式动作，输入 API id 和 key
  4. 开始享受每月 1000 次的免费数学公式识别吧
