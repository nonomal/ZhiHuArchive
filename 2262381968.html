问题描述

来了来了，渐进阅读是 SuperMemo 的一大特色，不得不品尝～

但是一本书实在太长，不仅不方便导入，放在一张摘录卡片中也难以控制阅读进度，还不易精准挑选自己想要阅读的内容。

这里就简单介绍一下，如何将一本书导入到 SuperMemo 中，然后将其按章节分割，方便渐进阅读。

导入书本

注意：这里介绍 EPUB 等电子书格式，PDF 还是另请高明了。

首先，我推荐下载 Calibre 用来查看电子书：

calibre - Download calibre (calibre-ebook.com)

然后用 calibre 打开你的电子书：

在电子书查看器中右键空白，设置首选项：

设置页面布局：

开启流式布局（这是最关键的一步！记得点确定）

然后，我们就可以开始按章节复制了！

先到你想要阅读的章节，

Ctrl + A 全选本章全部内容，然后 Ctrl + C 复制：

到 SuperMemo 按快捷键 Ctrl + N，就可以将整章内容粘贴进去了：

分割章节

一章的内容还是有点太多，我们可以用 SuperMemo 自带的分割文章功能：

勾选 custom string 作为我们的分割线标准

输入 <H2 ，点击确定，就可以按二级标题对章节进行分割了：

补充

如果只是分割章节的话，SuperMemo 给你安排阅读的顺序会有随机性。如果你要阅读的书，前后章节有非常强的依赖关系，必须看懂前面的章节才能阅读后续内容，我建议再参考一下这篇回答：
